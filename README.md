Muitas instituições possuem problemas como:

1.Falta de organização das ideias de melhoria tecnológica
2.Sugestões de alunos e funcionários não são registradas ou analisadas
3.Planejamento de redes de computadores feito sem colaboração ou documentação adequada
4.Ausência de um sistema que permita priorizar e avaliar projetos de infraestrutura

Consequências

1.Infraestrutura de rede mal planejada
2.Falta de inovação tecnológica
3.Desperdício de recursos
4.Baixa participação da comunidade institucional

O IdeaNet resolve isso criando um ambiente estruturado para gestão de ideias e planejamento de redes.
O sistema será voltado para instituições educacionais como:

1.Escolas
2.Faculdades
3.Universidades
4.Instituições técnicas

O Produto Mínimo Viável (MVP) do sistema terá:

1. Cadastro e Login de Usuários
Autenticação de usuários
Diferentes níveis de acesso

2. Registro de Ideias aonde os usuários podem cadastrar ideias como:

Expansão da rede Wi-Fi
Criação de VLANs
Implantação de servidores
Modernização da infraestrutura

3. Sistema de Avaliação
Votos nas ideias
Comentários
Ranking de melhores propostas

4. Gestão de Projetos de Rede
Transformar ideias aprovadas em projetos
Planejamento da rede
Documentação técnica

5. Painel Administrativo
Aprovação de ideias
Gestão de usuários
Acompanhamento de projetos
